## Appendix — Boomi University Learning Path (correct order)

Sequenced from Boomi's official **Recommended Training Paths** chart (Boomi Community, updated December 2025). Everyone starts with *Getting Started with the Boomi Platform → Integration Essentials*, then follows their role track. For a Presales SC, the Integration Developer → API Management spine plus the Boomi AI branch is the relevant path:

| Phase | Course | Certification |
|-------|--------|---------------|
| Pre-hire ✓ | Getting Started with the Boomi Platform | — |
| Pre-hire ● | Integration Essentials | — |
| 0–30 | Associate Integration Developer | Associate Developer Certification |
| 0–30 | Using AI Agents: Boomi Answers *(Boomi AI branch)* | — |
| 30–60 | Professional Integration Developer | Professional Developer Certification |
| 30–60 | Building an AI Agent with Agentstudio *(Boomi AI branch)* | — |
| 60–180 | Professional API Design | Professional API Design Certification |
| 60–180 | Professional API Management | Professional API Management Certification |
| 60–180 (stretch) | Associate Integration Architect | Associate Integration Architect Certification |

*Note: "Professional Developer", "Advanced Developer", "AgentStudio Fundamentals/Advanced" and "Professional Architect" are not Boomi University course names — the official ladder uses Associate/Professional tiers per track as above. Source: community.boomi.com Recommended Training Paths (Dec 2025).*
